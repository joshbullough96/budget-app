# Budget App

A local desktop budget application inspired by YNAB, built with Electron and JavaScript.

## Features
- Account management
- Category creation
- Transaction entry
- Monthly budget allocation with zero-based budgeting
- Sync with Google Sheets

## Setup
1. Clone the repo
2. Run `npm install`
3. Set up Google Sheets API credentials
4. Run `npm start`

## Development
- Main process: `main.js`
- Renderer: `src/renderer.js`
- Models: `src/models/`
- Services: `src/services/`
